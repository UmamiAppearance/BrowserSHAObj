BrowserSHAObj - Live Examples

Here you can find examples to demonstrate many features of BrowserSHAObj. You can run the code and also change it.
(Also check out the Online SHA Checksum Calculator), to see BrowserSHAObj operate in a real application.

In every example you have global access to the BrowserSHAObj - constructor.

☛ shaObj.update(a); shaObj.update(b) is in many occasions equivalent to shaObj.update(a+b)
